# Neovim config

## Requirements

- (fd)[https://github.com/sharkdp/fd]
- (ripgrep)[https://github.com/BurntSushi/ripgrep]

## Installation

```
cd ~/.config
git clone https://github.com/youginil/nvim.git
cd ~/.config/nvim
nvim init.lua
:PackerSync
```
